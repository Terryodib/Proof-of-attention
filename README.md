# Proof-of-attention

Proof of Attention is a Web3 intelligence tool that analyzes crypto tokens to detect whether market attention is real or artificially driven.

It evaluates onchain activity, wallet behavior, and social signals to produce clear risk and trust scores for any token.

What it does
Analyzes any token by contract address or symbol
Tracks smart money activity and wallet flows
Detects coordinated hype and manipulation patterns
Measures narrative strength and growth quality
Converts complex data into simple AI explanations
Key Scores
Attention Score
Organic Growth Score
Manipulation Risk Score
Smart Money Score
Narrative Strength Score

Each score comes with a short explanation that is easy to understand.

Core Features
Token intelligence dashboard
Wallet behavior tracking
Social signal analysis from X and Telegram patterns
Narrative analysis and comparison with past cycles
Alerts for abnormal activity and hype spikes
Tech Stack

Tailwind CSS

Node.js API routes
Supabase
OpenAI API
Ethers.js

Goal

To help users understand whether crypto attention is real or manufactured, and to reduce exposure to misleading hype cycles in the market.
